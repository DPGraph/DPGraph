Overview of Empirical Findings

There are four input variables that we can adjust when calculating the output of graph-DP algorithms. These four variables are:
  • Algorithm
  • Dataset
  • Query
  • Epsilon
It's interesting to investigate the effect of each of these variables on the accuracy of output. More specifically, fixing three of the four variables, we may be interested in whether or not the fourth variable affects the results, and if yes, how does it do so. We aim to address these questions from the empirical findings of our experiments. To see more details for each variable, click on the corresponding titles.

Algorithm:

Q: Is there one algorithm that outperforms the rest, across various input settings?

A: For degree distribution, no single algorithm dominates all others. For subgraph counting, Ladder is the dominant algorithm.


Dataset:

Q: Do graph-DP algorithms’ performances vary on graph datasets with different shapes?

A: Yes, graph datasets' shapes have an impact on algorithms' performances.


Query:

Q: (For SGC queries) Do graph-DP algorithms’ performances vary on different queries?

A: Yes, queries have an impact on graph-DP algorithms' performances.


Epsilon:

Q: Do graph-DP algorithms' performances vary on different epsilon values?

A: Yes. As we expected, algorithms' performances increases as epsilon increases.
